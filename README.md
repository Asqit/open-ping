# OpenPing

**Simple self-contained uptime pinger in Go!**

OpenPing is a lightweight utility that periodically checks the availability of your URLs and logs results locally. No UI, no sign-ups, no hosting hassle - single binary for your raspberry :D

## Install

download or clone the repository
```bash
$ git clone https://github.com/asqit/open-ping
$ cd open-ping
```

compile or run

```bash
$ templ generate # generates source code for templ UI
$ go run . # hot running the project
$ go build # compile the project.
```
